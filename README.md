# SystemyInternetowe_Projekt

Portal z grami przeglądarkowym stworzony jako projekt zaliczeniowy kursu Projektowanie i programowanie systemów internetowych I. Użytkownicy mogą rejestrować się, grać w gry zaimplementowane w JavaScript, śledzić swoje wyniki na globalnych tablicach wyników oraz tworzyć nowe wątki dyskusyjne na temat dostępnych gier.
